# MarmoAid 产品战略与技术护城河深度复盘

## 1. MarmoAid 核心功能全景图 (功能矩阵)

MarmoAid 是一个“非破坏性 AI 语义图像工作站”，功能矩阵如下：

### 核心视觉引擎 (Vision Engine)
*   **多模态生成与编辑**：文生图、图生图、局部重绘 (Inpainting)、扩图 (Outpainting)、画质超分重建 (4K Upscaling)。
*   **动态视频生成 (Video Generation)**：静态图像转动态视频（接入 Veo 模型）。
*   **多视角生成 (Multiview Generation)**：生成人物/商品/建筑的正、侧、俯三视图，保持材质和光影一致性。
*   **智能重光照 (Relighting)**：重新计算场景光影。
*   **视觉搜索与溯源 (Visual Search)**：识别家具/商品，生成搜索关键词，检索真实商品信息。

### 非破坏性语义图层系统 (Semantic Layer System)
*   **语义图层拆解 (Layer Explosion)**：将 2D 图像自动拆解为 3-8 个逻辑图层。
*   **智能抠图与边缘净化**：结合 Chroma Key 动态判定、边缘泛洪空洞检测 (Hole Detection Fallback) 和 Alpha 通道形态学膨胀，实现发丝级、半透明物体提取。
*   **无痕背景修补 (Clean Plate Generation)**：提取前景后自动无痕修补背景。
*   **图层级精准重绘**：锁定物体 3D 透视、体积和光影，仅对单一图层进行材质替换或重绘。
*   **图层版本控制**：支持单个图层的历史版本回溯 (Undo) 和重置 (Reset)。

### AI 自动化工作流引擎 (Node-based Workflow Engine)
*   **自然语言转 DAG 图 (Plan Graph)**：一句话自动转化为多节点工作流。
*   **场景化全案预设**：内置室内设计、电商上新、IP 孵化全案。

### 多智能体与协作系统 (Multi-Agent System)
*   **专家智能体评审 (Agent Analysis)**：描述、批判、创意专家进行专业点评，返回坐标级修改建议。
*   **智能提示词推荐**：动态生成下一步操作建议。

### 无限画布与空间交互 (Infinite Canvas & Spatial UI)
*   **Chat + Canvas 混合形态**：无限工作台，支持拖拽、缩放、平移、框选。
*   **血统连线 (Genealogy Lines)**：可视化展示图像演变史。
*   **资产库**：一键收藏生成的图层/资产，支持云端同步。

---

## 2. 核心壁垒与护城河 (The Moat)

1.  **技术壁垒：语义级处理**：从“像素级”覆盖转向“语义级”拆解与重组，实现非破坏性编辑。
2.  **交互壁垒：Chat 与 Node-Graph 的融合**：自然语言驱动的隐形节点流，降低复杂工作流门槛。
3.  **场景化工程壁垒**：针对室内设计、电商等垂直领域定制的 Prompt 约束和自动化流程。

## 4. 商业战略调整 (Pivot)

基于残酷的市场拆解，MarmoAid 战略转向：

*   **定位调整**：从“AI 设计工具”转向 **「AI Spatial Asset Editor (空间资产编辑器)」**。
*   **核心价值**：把任意空间/家具图片转化为可编辑资产，并无缝重组到真实空间。
*   **战略聚焦**：All in **室内设计 / 软装 / 空间视觉** 领域。
*   **行动指南**：
    1.  **Code Freeze**：停止开发边缘功能（视频、多专家 Agent 等）。
    2.  **强化核心**：极致打磨分层拆解、抠图边缘、资产复用。
    3.  **PMF 验证**：停止闭门造车，寻找 10 个真实设计师进行深度访谈，验证工作流痛点与付费意愿。

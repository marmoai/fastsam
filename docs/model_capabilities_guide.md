# Gemini 3 系列最新模型能力与 Prompt 优化指南 (2026.03)

本指南旨在帮助您深入了解当前应用中集成的最新一代 Gemini 3 系列模型（内部代号 Banana Pro/Banana 2 等）的核心能力、深度挖掘潜力以及如何通过优化 Prompt 来最大化其性能。

---

## 1. 核心模型矩阵及其分工

在当前应用中，我们采用了“多模型协同”的最优架构：

| 模型名称 | 应用场景 | 核心优势 |
| :--- | :--- | :--- |
| **Gemini 3.1 Pro Preview** | 文本逻辑、流程策划、复杂推理 | 极强的逻辑推理能力、200万超长上下文、精准的工具调用。 |
| **Gemini 3.1 Flash Image Preview** | 看图改图、材质替换、局部重绘 | 视觉理解力极强，支持基于 Mask（遮罩）的精准空间编辑。 |
| **Imagen 4.0 Generate 001** | 纯文本生图 (Text-to-Image) | 顶级的画质细节、照片级写实度、极高的审美上限。 |
| **Veo 3.1 Generate Preview** | 高清视频生成 (1080p) | 电影级画质、长达 7 秒的连贯动作、极佳的时间一致性。 |

---

## 2. 深度挖掘能力：Gemini 3.1 Pro (The Brain)

### 核心能力：
*   **超长上下文 (Long Context)**：支持高达 200 万个 Token。这意味着您可以一次性向它“喂”入几百张图片、几小时的视频或几千页的文档，它能进行跨模态的深度关联分析。
*   **复杂逻辑推理 (Reasoning)**：在“流程策划”节点中，它不仅能理解您的指令，还能预测可能出现的逻辑冲突并自动优化步骤。
*   **工具调用 (Tool Use)**：原生集成 Google Search（实时搜索）和 Google Maps（地理位置感知），能获取最新资讯并结合地图数据进行决策。

### Prompt 优化技巧：
*   **角色设定**：给它一个具体的专家身份（如“资深室内设计师”、“3D 渲染专家”）。
*   **思维链 (CoT)**：在 Prompt 中加入“请一步步思考”或“先分析原图结构，再制定修改计划”，能显著提升复杂任务的成功率。
*   **少样本提示 (Few-Shot)**：提供 1-2 个“输入 -> 输出”的例子，它能迅速模仿您的风格。

---

## 3. 视觉编辑能力：Gemini 3.1 Flash Image (The Artist)

### 核心能力：
*   **空间感知 (Spatial Understanding)**：它能精准识别图片中物体的 3D 结构、光影方向和遮挡关系。
*   **遮罩编辑 (Inpainting/Outpainting)**：配合我们现在的“双图引导（原图+黑白蒙版）”方案，它能实现像素级的局部修改，而完全不破坏框外区域。
*   **材质迁移 (Material Transfer)**：能完美地将参考图的纹理“贴”到原图的物体上，并自动适配原图的光影。

### Prompt 优化技巧：
*   **结构约束**：使用“保持原始物体的 3D 形状和骨架不变”来防止形变。
*   **光影同步**：强调“继承原图的光照方向、高光和阴影”，使修改后的部分看起来毫无违和感。
*   **细节描述**：不要只说“换成大理石”，要说“换成带有深色纹理的、高光反射明显的白色大理石”。

---

## 4. 极致生图：Imagen 4.0 (The Photographer)

### 核心能力：
*   **照片级写实 (Photorealism)**：在处理皮肤纹理、金属反射、自然光效方面达到了行业顶尖水平。
*   **审美理解**：能理解复杂的艺术风格（如“赛博朋克”、“极简主义”、“印象派”）。

### Prompt 优化技巧：
*   **相机参数**：加入“Shot on 35mm lens, f/1.8, cinematic lighting”等摄影术语，能大幅提升画面质感。
*   **负面约束（隐含）**：虽然 Imagen 4 不直接支持负面词，但通过正面描述“Clean background, sharp focus, no distortion”可以达到同样效果。

---

## 5. 电影级视频：Veo 3.1 (The Director)

### 核心能力：
*   **1080p 高清**：支持生成 1080p 分辨率的视频，细节清晰度远超上一代。
*   **动态一致性**：物体在移动过程中不会发生诡异的形变或消失。

### Prompt 优化技巧：
*   **动作描述**：使用动词丰富的短语，如“The camera slowly pans right while the character is running through the rain”。
*   **氛围设定**：描述天气、光效和情绪，如“Moody atmosphere, neon lights reflecting on wet pavement”。

---

## 6. 针对您的测试建议：

1.  **测试“材质替换”时**：尝试给出一个光影非常复杂的场景（比如有强侧光或倒影），看 Gemini 3.1 Flash 是否能完美保留这些光影。
2.  **测试“流程策划”时**：给出一个包含 5 个以上步骤的复杂任务，看 Gemini 3.1 Pro 是否能逻辑自洽地拆解并执行。
3.  **测试“Imagen 4”时**：尝试生成包含文字或精细手部动作的图片，这是检验生图模型能力的“试金石”。
